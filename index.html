Über mich

Hey, mein Name ist Fabian und ich beschäftige mich in meiner Freizeit mit APIs, Webdesign, Musik, Kampfkunst und Storytelling. Seit zwei Jahren lerne ich JavaScript und entwickle eigene Projekte, darunter einen Bot für meinen Discord-Server mit über 130 Mitgliedern.

Außerdem spiele ich Keyboard in der Punk-Rock-Coverband Noise Education. Nachdem wir anfangs nur auf Schulkonzerten gespielt haben, treten wir mittlerweile regelmäßig auf Festivals und Veranstaltungen außerhalb der Schule auf, wie dem Dorfsound Festival oder bei Events in Gauting.

Als kreativen Ausgleich leite ich als Dungeon Master eine eigene D&D-Kampagne für eine Gruppe von sieben Spielern in einer selbst entworfenen Fantasy-Welt. Zudem mache ich Taekwondo im 4. Kup.

Kekse Bot

Der Kekse-Bot ist ein maßgeschneidertes, funktionsreiches
Discord.js-Bot-System (v14), das speziell für den Discord Server
Keksegang entwickelt wurde. Er kombiniert ein tiefgehendes
Wirtschafts- und Casino-System mit einem sicheren,
administrativen Web-Dashboard inklusive
Progressive Web App (PWA) Unterstützung.

zu erstellen. Eine API sucht im Hintergrund Informationen
